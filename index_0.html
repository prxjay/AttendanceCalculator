Attendance Calculator

How to Use?

1. Inputting Data:

Select the minimum attendance percentage required from the dropdown.

Enter the total number of classes in the "Total Classes" field.

Enter the number of classes attended in the "Classes Attended" field.

2. Calculating Attendance:

Click on the "Calculate" button to compute the attendance details.

The result will be displayed below, showing either the number of classes to attend to meet the required percentage or the number of classes that can be skipped if the attendance requirement is already met.

3. Downloading Attendance Data:

After calculating the attendance, you can download the attendance data.

Click on the "Download Attendance" button to save the attendance details as a JSON file.
